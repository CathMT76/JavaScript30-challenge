## Key concepts

### CSS variables

The use of CSS variables to assign a value across multiple elements and properties throughout the document.

### Enable live preview: `change` and `mousemove` event handlers

Use of `change` and `mousemove` event handlers to immediately reflect the effects of the new input on the sample images - enabling a "live preview" effect to the user.
